Hardware Parts

Learn every component inside a computer—from the CPU and motherboard to RAM, storage, and power supply.

Computer Hardware Components

Learn the function of each hardware component inside a computer.

Motherboard

Connects all computer components and allows them to communicate.

CPU (Processor)

Executes instructions and performs calculations.

RAM

Temporarily stores data used by running programs.

Graphics Card (GPU)

Processes graphics for games, videos, and applications.

SSD

Stores the operating system, software, and files with fast access.

Hard Disk Drive

Provides long-term storage for files and applications.

Power Supply (PSU)

Supplies electrical power to every component in the computer.

CPU Cooler

Removes heat from the processor to prevent overheating.

Case Fan

Improves airflow and helps cool the computer.

Computer Case

Holds and protects all computer hardware.

CMOS Battery

Maintains BIOS settings and the system clock.

Thermal Paste

Improves heat transfer between the CPU and the cooler.

Network Card

Connects the computer to a wired network.

Wi-Fi Adapter

Connects the computer to a wireless network.

Sound Card

Processes audio input and output.

BIOS / UEFI Chip

Starts the computer and initializes the hardware.
